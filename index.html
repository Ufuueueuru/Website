Use the @lt;h1@gt; tag to do a large heading.

The numbers can go from 1-6 after the h for smaller headings.

This is @lt;h3@gt;

This is @lt;h4@gt;

This is @lt;h5@gt;
This is @lt;h6@gt;

This is a paragraph! Use the <p@gt; tag to do it!

Lists can be made with several @lt;li@gt; inside of @lt;ul@gt; or @lt;ol@gt;

  • <b> - Bold text
  • <i> - Italic text
  • <sub> - Subscript text
  • <sup> - Superscript text
